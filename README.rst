=====
PyEnv
=====

Installation scripts used to bring a fresh installation to a python development environment.

Usage
-----

.. code:: shell

        wget -O - https://raw.githubusercontent.com/AndrewRPorter/pyenv/master/install.sh | bash

Included Software
-----------------

- atom
- pip3
- vim
- git
- unzip
- gnome-terminal
- tldr
- bat
- exa

Python Packages
---------------

- requests
- django
- PYYaml
- boto3
- gunicorn
- jinaj2
- black
- virtualenv
